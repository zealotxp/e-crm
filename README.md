# CRM系统 - Arco Design Vue

基于 Vue 3 + Arco Design Vue 开发的软件开发外包企业 CRM 系统。

## 技术栈

- **框架**: Vue 3 (Composition API)
- **UI组件库**: Arco Design Vue 2.55+
- **构建工具**: Vite 5
- **状态管理**: Pinia 2
- **路由**: Vue Router 4
- **图表**: ECharts 5
- **HTTP客户端**: Axios
- **日期处理**: Day.js

## 功能模块

### 1. 工作台 (Dashboard)
- 数据概览（签约金额、回款金额、新增客户、进行中项目）
- 销售漏斗图表
- 回款趋势图
- 待办事项
- 最近动态

### 2. 客户管理 (Customer)
- 客户列表（支持筛选、搜索、分页）
- 客户详情（基本信息、联系人、跟进记录、商机、报价、合同、项目）
- 客户新建/编辑
- 联系人管理
- 跟进记录
- 公海池

### 3. 商机管理 (Opportunity)
- 商机列表/看板双视图
- 销售漏斗（线索→接触→需求确认→方案→报价→谈判→赢单/输单）
- 商机推进
- 预计金额和成交日期
- 赢单率分析

### 4. 报价管理 (Quotation)
- 报价单列表
- 报价明细
- 报价审批
- 报价导出
- 报价转合同

### 5. 合同管理 (Contract)
- 合同列表
- 付款条款
- 合同审批
- 合同到期提醒

### 6. 项目管理 (Project)
- 项目卡片/列表视图
- 项目进度跟踪
- 项目团队
- 项目状态管理

### 7. 任务管理 (Task)
- 看板视图（待办/进行中/已完成/已取消）
- 任务拖拽
- 优先级管理
- 工时记录

### 8. 回款管理 (Payment)
- 回款计划
- 回款记录
- 逾期提醒
- 账龄分析

### 9. 开票管理 (Invoice)
- 开票申请
- 开票审批
- 快递跟踪

### 10. 工单管理 (Ticket)
- 工单列表
- 工单分类（Bug/需求/咨询/投诉）
- 优先级管理
- 工单处理

### 11. 报表中心 (Report)
- 销售报表
- 项目报表
- 财务报表

### 12. 系统设置 (Setting)
- 用户管理
- 角色权限
- 组织架构
- 系统参数

## 项目结构

```
crm-arco/
├── public/                 # 静态资源
├── src/
│   ├── api/               # API接口
│   ├── assets/            # 资源文件
│   ├── components/        # 公共组件
│   ├── layouts/           # 布局组件
│   ├── router/            # 路由配置
│   ├── stores/            # Pinia状态管理
│   ├── utils/             # 工具函数
│   ├── views/             # 页面视图
│   │   ├── login/         # 登录页
│   │   ├── dashboard/     # 工作台
│   │   ├── customer/      # 客户管理
│   │   ├── opportunity/   # 商机管理
│   │   ├── quotation/     # 报价管理
│   │   ├── contract/      # 合同管理
│   │   ├── project/       # 项目管理
│   │   ├── task/          # 任务管理
│   │   ├── payment/       # 回款管理
│   │   ├── invoice/       # 开票管理
│   │   ├── ticket/        # 工单管理
│   │   ├── report/        # 报表中心
│   │   ├── setting/       # 系统设置
│   │   └── error/         # 错误页面
│   ├── App.vue            # 根组件
│   └── main.js            # 入口文件
├── index.html
├── package.json
├── vite.config.js
└── README.md
```

## 安装运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产环境
npm run build

# 预览生产构建
npm run preview
```
